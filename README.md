# abstract-gan

A simple **DCGAN** implemented in **PyTorch** to generate abstract images. [Description generated by ChatGPT]

---

## Features

* **Custom Dataset & Augmentation:** Flexible PyTorch Dataset with random flips, rotations, and resizing
* **DCGAN Architecture:** ConvTranspose layers with BatchNorm, LeakyReLU, and Tanh for stable image generation
* **Latent Space Exploration:** Maps random vectors to realistic images
* **Training Stability:** Separate optimizers for Generator and Discriminator, Adam with Î²1=0.5, real/fake label handling
* **Progress Tracking:** Fixed noise vectors allow monitoring Generator improvements over epochs
* **Real-Time Visualization:** Matplotlib previews generated images during training
* **GPU Acceleration:** Automatically uses CUDA if available for faster training
* **Modular & Configurable:** Easily adjust batch size, epochs, image size, and save intervals
